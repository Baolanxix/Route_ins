# Field Route Navigator v22

Bản này chỉnh từ v21/v20:

- Bỏ nút **Bỏ qua đoạn** và **Bỏ qua 300m**.
- Thêm nút **Tiếp tục đi**.
- Khi bấm **Tiếp tục đi**, app dùng GPS hiện tại để bỏ qua các đoạn chưa đi ở phía sau/kế hoạch cũ, rồi tiếp tục chỉ dẫn từ vị trí hiện tại.
- Giữ import KMZ/KML thủ công.
- Giữ OSM smart route, hiển thị mét còn lại, GPS trượt mượt, mũi tên lệch phải.
